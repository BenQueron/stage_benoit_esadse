Le jeu-vidéo donné aux étudiants

Un jeu vidéo comme support de cours! En amont du workshop, deux environnements de jeu ont été créés avec l'outil. Ces environnements ont ensuite été donnés aux étudiants en introduction de l'atelier, permettant d'annoncer la production avec une phase de jau (gameplay). Vous avez ici les deux versions de ce jeu-test qui srvent de tutoriels aux mécaniques de création de l'outil. L'objectif est pour les étudiants d'associer une progression dans les niveaux du jeu avec un dégré de complexité de réalisation. Le level design suit ici une gradation d'usages de l'outil en partant des interactions par les flèches du clavier jusqu'à des transitions invisibles permises par des variables programmées. Ces jeux ont été mis en ligne sur github et ont été téléchargés par les étudiants. Bitsy permet ensuite de modifier directement les jeux et donc de s'assurer que tous les étudiants partagent le même environnement de création numérique.
